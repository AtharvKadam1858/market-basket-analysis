# Market Basket Analytics Dashboard

## Overview

This project implements Market Basket Analysis using Apriori Algorithm to identify product associations and cross-selling opportunities.

The system processes over 100,000 transactions and provides interactive insights through a Streamlit dashboard.

---

## Features

- Market Basket Analysis
- Apriori Algorithm
- SQL Database Integration
- Interactive Dashboard
- Product Recommendation Engine
- Business Insights
- Cross Sell Analysis

---

## Tech Stack

Python  
SQL (SQLite)  
Streamlit  
Apriori Algorithm  
Plotly

---

## Dataset

100,000+ simulated retail transactions.
